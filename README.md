# Beef-XSS
Beef-XSS installation file for parrot users
if you want to innstall the requirements for Beef-XSS install this file to get it into your parrot Security.
Commands are

  -> chmod +x install.sh

give the executable permissions to the install.sh file.
and you are must be a root user to install this requirements if you install this file as a normal user, there is a chance you may get errors for sure.
now execute the commands
  
  -> sudo ./install .sh
  
after that requirements installtion will be done.
now you need to type the following command to install Beef-XSS in your Parrot Security.

  -> apt install beef-xss
